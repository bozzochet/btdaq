DAQ, Data Acquisition software:

Usable via EPP or PCI. Compatible with TDR, JINF and  JINJ DSP programs as in July 2009 and usable with JLV1, NI-USB or manual trigger. The data can be written in AMS Block format or in binary standard one. Calibration can be written in AMS Block format or text one visible with SUMMARY on a pdf.

Enjoy!
